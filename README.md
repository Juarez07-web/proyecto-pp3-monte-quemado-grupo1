# Análisis de Gestión Presupuestaria - Municipalidad de Monte Quemado

## Problema
La Secretaría de Obras y Servicios Públicos presenta una mala imputación recurrente de gastos de personal en otras partidas presupuestarias, lo que dificulta la transparencia y planificación financiera.

## Objetivo
Reclasificar los gastos de personal correctamente y desarrollar un modelo de pronósticos para la planificación presupuestaria 2026.

## Metodología
1. **Limpieza de datos:** Procesamiento de 1,853 registros (Enero-Julio 2025)
2. **Reclasificación:** Aplicación de regla de negocio (pagos sin factura = personal)
3. **Modelado:** Implementación de modelo de tendencia lineal para proyección
4. **Presupuesto 2026:** Cálculo de proyección mensual (+58,9% vs 2025)

## Resultados
- **Presupuesto 2026 proyectado:** $371.1M
- **Gasto en personal identificado correctamente:** $104.75M
- **Modelo de pronósticos con tendencia:** +$954,836/mes
